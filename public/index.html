Projects

Field View App

I created this app with my team during my time as an IT intern at John Deere to assist farmers with the act of field scouting. The Java application allows users to drap and drop pins of data onto an image of the user's field. The app lets users input a picture of their own field or use Google Maps to view the area. In these information pins, users can store problem type, severity, text, and pictures. Data gets stored into a database every time there is an update to the app and is fetched everytime there is a login.
